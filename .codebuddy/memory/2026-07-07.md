## 导出改为后端直写（23:19）
用户质疑"首次必须选 Desktop 并记 IndexedDB"这一步是否必要，希望直接落到 `~/Desktop/剧本名_第x集_分镜_x`。
结论：纯前端因浏览器沙箱强制需要选文件夹；但本项目是本地 Node 全栈应用，改为**后端直写**可彻底免选文件夹。
实现：
- server.ts 新增 `POST /api/export-shot-assets`，接收 `{ folderName, files:[{name,dataUrl}] }`，用 `os.homedir()` 拼 `~/Desktop/<folderName>` 写盘。
- Detail.tsx 删除 downloadImage / IndexedDB 句柄逻辑，新增 urlToDataUrl（data: 直传，http 用 canvas 转 dataURL）；exportShotAssets 第三步改为 POST 给后端。
- 注意：改了 server.ts，**必须重启 `npm run dev`**（tsx 不会热重载后端）。
