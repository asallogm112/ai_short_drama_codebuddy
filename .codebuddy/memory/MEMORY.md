## ai_drama_studio 项目：分镜时间轴分段规则（核心约定）
用户明确要求视频分镜的时间轴分段采用「内容驱动、长短不一」策略，不是均匀网格、也不是固定模板。

- ❌ 不要：0-0.5/0.5-1 均匀切；也不要 0-3/3-6/6-10 大段（画面感太差）。
- ✅ 要：如 0-2 / 2-2.5 / 2.5-5 / 5-6 / 6-8 / 8-8.5 / 8.5-10 这种按动作强度智能切分。
- 统一的是「规则」不是「数字」：平缓交代/空镜 → 2-3秒长拍+远景/全景/侧拍；普通表演 → 1-2秒+中景；关键微动作/情绪转折 → 0.5-1秒短拍+近景/特写/低角度仰拍/俯拍。
- 每一段必须带：景别 + 拍摄角度 + 镜头运动 + 主体动作 + 光效/细节。
- 相邻段姿态连续（上段结尾=下段开头）。
- 具体起止秒数由 AI(DeepSeek) 按每个 shot 内容现生成，不套固定 schema。
- 素材头部(@R1_林晚 : @R1_林晚.jpg)与结尾话术("直接生成视频…无版权无真人")由代码在复制时拼接，不进 LLM 输出。
